Setup instructions: 
This project is a Node.js application with APIs. 
It includes a `package.json` file that lists all the dependencies required for the project. 
If the `node_modules` folder has been deleted or is missing, you can easily install all the necessary dependencies using `npm install`.

Navigate to the Project Directory:
cd yourproject

Run the following command to install all the dependencies listed in the package.json file:
npm install

Start the Application:
nodemon index.js
or 
node index.js
